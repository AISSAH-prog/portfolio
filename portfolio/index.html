Projet Cloud Computing — DataCenter
Objectif du projet

Transformer les PC d'une salle informatique en une infrastructure DataCenter complète avec déploiement automatisé, supervision en temps réel et sécurisation réseau.

Technologies utilisées
  • Vagrant : Déploiement automatisé des machines virtuelles
  • Ansible : Configuration et gestion automatisée (IaC)
  • LAMP Stack : Services web (Linux, Apache, MySQL, PHP)
  • Tomcat : Serveur d'applications Java
  • Zabbix : Supervision et monitoring de l'infrastructure
  • OPNsense : Firewall et sécurisation réseau
  • Load Balancer : Répartition de charge et haute disponibilité
